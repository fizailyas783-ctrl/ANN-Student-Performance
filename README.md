# 🎓 ANN-Based Student Performance Evaluator

A complete Artificial Neural Network project that predicts student **Pass/Fail** outcomes based on academic features.

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://your-app-name.streamlit.app)

---

## 📂 Project Structure

```
student_ann_project/
│
├── dataset.xlsx       ← Student performance dataset (600 records)
├── train_ann.py       ← Tasks 1–8: explore, preprocess, train, evaluate, save
├── predict.py         ← Task 7: reusable evaluate_student() + CLI interface
├── app.py             ← Task 9: Streamlit web UI
├── model.joblib       ← Saved trained ANN (generated by train_ann.py)
├── scaler.joblib      ← Saved StandardScaler (generated by train_ann.py)
├── requirements.txt   ← Python dependencies
└── README.md
```

## 🚀 Quick Start

### 1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/student-ann-evaluator.git
cd student-ann-evaluator
```

### 2. Create virtual environment
```bash
python -m venv venv
venv\Scripts\activate        # Windows
source venv/bin/activate     # Mac/Linux
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Train the model
```bash
python train_ann.py
```
This will:
- Explore and preprocess the dataset
- Train the ANN
- Print accuracy, confusion matrix, classification report
- Save `model.joblib` and `scaler.joblib`
- Save `training_report.png`

### 5. Run the Streamlit app
```bash
streamlit run app.py
```

### 6. (Optional) Run CLI predictor
```bash
python predict.py
```

## 📊 Dataset Features

| Column       | Description                        |
|--------------|------------------------------------|
| attendance   | Attendance percentage (0–100)      |
| assignment   | Assignment marks                   |
| quiz         | Quiz marks                         |
| mid          | Mid-term exam marks                |
| study_hours  | Average daily study hours          |

## 📝 Key Concepts

**What is ANN?**  
An Artificial Neural Network is a mathematical model inspired by the human brain. It learns patterns by adjusting weights through gradient descent.

**Why StandardScaler?**  
Features have different ranges. Scaling ensures equal contribution of all features during gradient-based training.

**evaluate_student() function:**
```python
from predict import evaluate_student

result = evaluate_student(
    attendance=85,
    assignment=90,
    quiz=78,
    mid=70,
    study_hours=8
)
print(result)
# {'result': 1, 'label': 'Pass ✅', 'probability': {...}, 'performance': '🟢 High Performance'}
```
